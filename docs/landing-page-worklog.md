## 🔧 기술적 변경 사항

### 추가된 React 훅/기능
- `useState(isScrolled)` - 스크롤 상태 감지
- `useEffect` - 스크롤 이벤트 리스너
- `scrollToSection()` - 스무스 스크롤 함수
- 마우스 드래그 핸들러 (onMouseDown/Move/Up/Leave)

### 추가된 Tailwind 클래스
- `scroll-mt-20` - 네비 높이 고려 스크롤 마진
- `select-none` - 텍스트 선택 방지
- `cursor-grab / active:cursor-grabbing` - 드래그 커서

### 네비게이션 링크 데이터
```tsx
const navLinks = [
  { label: '문제 해결', href: '#problem-section' },
  { label: 'AI 심사위원회', href: '#makers-section' },
  { label: 'M.A.K.E.R.S', href: '#makers-committee' },
  { label: '맞춤 지원', href: '#business-category' },
```
