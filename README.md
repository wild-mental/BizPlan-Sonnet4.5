## 🎪 M.A.K.E.R.S AI 평가위원회

정부지원사업 평가의 6가지 핵심 영역을 전담하는 AI 심사위원단

| 위원 | 영역 | 한국어 | 평가 항목 |
|------|------|--------|----------|
| **M** | Marketability | 시장성 | 시장 규모, 고객 수요, 경쟁 상황, 타깃 시장 분석 |
| **A** | Ability | 수행능력 | 팀 구성, 창업자 역량, 실행 가능성, 인프라 |
| **K** | Key Technology | 핵심기술 | 기술 혁신성, 차별화, 지식재산권, 기술 보호 |
| **E** | Economics | 경제성 | 매출/손익 계획, 자금 조달, 투자 회수, 재무 건전성 |
| **R** | Realization | 실현가능성 | 사업 추진 일정, 단계별 실행 계획, 리스크 관리 |
| **S** | Social Value | 사회적가치 | 일자리 창출, 지역 균형, ESG, 정부 정책 방향 |

## 🛠 기술 스택

- **Framework:** React 19 + Vite 7
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Custom Glassmorphism
- **State:** Zustand
- **Routing:** React Router DOM v7
- **Icons:** Lucide React
- **Charts:** Recharts
